## Image Guidelines

### Recommended Sizes

**Profile Photos:**
- Square format: 800x800px minimum
- Format: JPG or PNG
- Max file size: 500KB

**Project Logos:**
- Square format: 400x400px
- Format: PNG with transparent background
- Max file size: 200KB

**Book Covers:**
- Portrait format: 600x800px recommended
- Format: JPG or PNG
- Max file size: 300KB each

**Blog Thumbnails:**
- Landscape format: 1200x630px (optimal for social sharing)
- Format: JPG
- Max file size: 500KB each

**Speaking/Event Photos:**
- Landscape format: 1200x800px minimum
- Format: JPG
- Max file size: 500KB

### Optimization Tips

1. **Compress your images** before adding them
   - Use tools like TinyPNG, Squoosh, or ImageOptim
   - Target: High quality with smallest file size

2. **Use appropriate formats**
   - JPG for photos
   - PNG for logos/graphics with transparency
   - WebP for better compression (Next.js will auto-convert)

3. **Responsive images**
   - Provide high-resolution images
   - Next.js Image component will auto-optimize

## How to Add Images

1. Save your images to this folder
2. Name them exactly as listed above (or update the component files)
3. The website will automatically display them
